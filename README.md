# Courier-Service
a command line application for courier service
## Requirements
- node.js
## Getting Started
This repositories has two applications which are located in two different folders(`problem01` and `problem02`).

1. Install packages

    ```bash
      npm install
    ```

2. Start the application
   ```bash
      npm start
    ```
3. Test the application
   ```bash
      npm test
    ```
